# Contributing

ご協力ありがとうございます。誤情報を利用者へ届けないことを最優先にします。

## 基本ルール

- 変更はプルリクエストで提案してください。
- 一つのプルリクエストには、できるだけ一つの目的だけを含めてください。
- UI変更はスマートフォン表示、キーボード操作、読みやすさを確認してください。
- データ変更は [DATA_SOURCES.md](DATA_SOURCES.md) の確認項目をすべて満たしてください。
- 推測やAI出力だけを根拠に、事業者情報を追加・変更しないでください。

## ライセンスへの同意

プルリクエストを提出することで、あなたは次を確認したものとします。

1. 提出内容を提供する権利があること
2. 提出内容を本プロジェクトの `AGPL-3.0-only` のもとで提供すること
3. 第三者のデータや素材を含む場合、出典と適用条件を明記したこと

マージには `@moshimur` の確認が必要です。
