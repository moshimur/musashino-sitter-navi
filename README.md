# 武蔵野シッターナビ

武蔵野市の「ベビーシッターによる一時預かり利用支援事業」を利用したい保護者が、東京都の認定事業者を条件で絞り込み、連絡先や公式情報を確認するためのスマートフォン向けWebアプリです。

夜間など、保護者が疲れている状況でも迷わず操作できるよう、大きな文字、広いタップ領域、落ち着いた配色を採用しています。

## 主な機能

- 東京都認定事業者37社の一覧表示
- 入会金・年会費、病児、夜間、当日予約、障害児対応などによる複数条件の絞り込み
- 住所または現在地を基準にした、公開事業所所在地までの参考距離表示
- 料金、最低利用時間、キャンセル料、電話番号、公式サイトの表示
- 個人プロフィール検索を公開している事業者への導線
- 武蔵野市の補助制度概要と注意事項の表示

## 使い方

`index.html`をブラウザで開いてください。HTML、CSS、JavaScriptと事業者JSONを1ファイルにまとめているため、ビルド作業は不要です。

`file://`で直接開いても事業者一覧と絞り込みを利用できます。住所検索にはインターネット接続が必要です。

Cloudflare Pagesなどの静的ホスティングにも配置できます。

## データの出典

掲載情報は、主に次の公的情報を基に整理しています。データ取得・確認日は2026年7月14日です。

- [東京都福祉局「ベビーシッター利用支援事業（一時預かり利用支援）認定事業者一覧」](https://www.fukushi.metro.tokyo.lg.jp/kodomo/hoiku/bs/itijiazukarijigyoushalist)
  - 認定事業者名、事業者数などの基礎情報
- [江戸川区「ベビーシッター利用支援事業（一時預かり利用支援）認定事業者一覧」比較資料](https://www.city.edogawa.tokyo.jp/documents/69753/ninteijigyosha_.pdf)
  - 料金、入会金・年会費、最低利用時間、キャンセル料、対応条件など
- [国税庁 法人番号公表サイト](https://www.houjin-bangou.nta.go.jp/)
  - 新規掲載事業者などの本店所在地確認（株式会社グロウキッズ：法人番号2011301030237）
- [武蔵野市「ベビーシッターによる一時預かり利用支援事業」](https://www.city.musashino.lg.jp/shussan_kodomo_kyoiku/kodomo_kosodate/hoikuen_yochien_kodomoen/kanrenservice/1053852.html)
  - 対象者、利用上限、助成上限額、申請上の注意事項
- [国土地理院 地理院地図](https://maps.gsi.go.jp/)
  - 公開事業所所在地の座標化と住所検索

各事業者の所在地は、上記に加えて自治体公開資料および各事業者の公式サイトも参照しています。

## データ構成

- `sitter-data.json`：東京都認定事業者37社、武蔵野市の制度情報、出典、免責事項
- `index.html`：`sitter-data.json`の内容をJSON構造を変えずに埋め込んだアプリ本体

## 重要な免責事項

本アプリは、武蔵野市、東京都、各掲載事業者が運営する公式サービスではありません。

- 掲載情報は2026年7月時点の情報です。料金、規約、対応地域、認定状況などは変更される場合があります。
- 距離は入力地点と各社が公開する本社・本店・支社・事業所等との直線距離です。サービス提供エリア、シッター個人の所在地、移動時間、派遣可否、現在の空き状況を示すものではありません。
- 本店所在地と実際のベビーシッター事業所が異なる場合があります。
- 江戸川区の資料に掲載された対応状況は、江戸川区での調査結果です。武蔵野市での対応可否を保証するものではありません。
- 個人シッターが東京都の補助対象要件を満たすか、希望日時に訪問できるかは、予約前に各事業者へ確認してください。
- 入会金、保険料、交通費、キャンセル料などは助成対象外となる場合があります。
- 住所検索に入力した情報はアプリ内に保存しませんが、検索のため国土地理院の住所検索サービスへ送信されます。

**最新情報は必ず各事業者・武蔵野市・東京都の公式ページでご確認ください。**
